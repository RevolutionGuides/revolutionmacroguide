# System Requirements

## Minimum Display Requirements

- **Resolution**: 1920x1080 (Windows) or 1366x768 (absolute minimum)
- **Scale**: 100% (no scaling)
- **Display Mode**: Standard LCD, no HDR
- **Input**: Keyboard and Mouse (no touchscreen support)

## Recommended Setup

- **Resolution**: 1920x1080 or higher
- **Refresh Rate**: 60+ FPS for optimal performance
- **GPU**: Dedicated graphics recommended for consistent frame rates
- **RAM**: 8GB minimum, 16GB recommended
- **Storage**: 1GB free space

## Operating Systems

- **Windows**: Windows 10/11 (64-bit)
- **macOS**: macOS 10.13 or later (Intel or Apple Silicon with Rosetta 2)

## Game Settings (Roblox BSS)

- Roblox language set to **English**
- Graphics quality optimized for your system
- Performance stats disabled during macro use
- Camera mode set to **Classic/Default**
- Text size set to **zero** for optimal recognition

## Network

- Stable internet connection required
- No VPN or proxy interference with Roblox connections
- Low latency recommended for macro responsiveness
