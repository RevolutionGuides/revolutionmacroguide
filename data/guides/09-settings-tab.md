# Settings Tab

The Settings Tab contains global macro configuration.

## General Settings

| Setting           | Purpose                    |
| ----------------- | -------------------------- |
| **Language**      | Select UI language         |
| **Theme**         | Dark/Light mode preference |
| **Auto-start**    | Launch macro on startup    |
| **Always on Top** | Keep window above others   |
| **Debug Mode**    | Enable detailed logging    |

## Display Settings

| Setting                 | Purpose                               |
| ----------------------- | ------------------------------------- |
| **Capture Mode**        | Screen capture method (Direct/Legacy) |
| **UI Scale**            | Adjust interface size                 |
| **Show Status Overlay** | Display stats on screen               |
| **Opacity**             | Window transparency level             |

## Performance Settings

| Setting          | Purpose                    |
| ---------------- | -------------------------- |
| **Frame Skip**   | Balance speed vs. accuracy |
| **CPU Priority** | Process priority level     |
| **Memory Limit** | Max RAM usage cap          |
| **Hotkeys**      | Custom keyboard shortcuts  |

## Advanced Configuration

### Capture Method

- **Direct Capture**: Faster, more accurate (recommended)
- **Legacy Monitor**: For compatibility issues

### Custom Hotkeys

Customize default key binds:

- START: F1 (changeable)
- PAUSE: F2 (changeable)
- STOP: F3 (changeable)

## Backup & Restore

- Export current settings to file
- Import previously saved settings
- Share configurations with community

## Reset Options

- Reset to defaults (careful - removes all customization)
- Reset individual sections
- Verify important settings before resetting

## Tips

- Experiment with settings in test sessions first
- Keep a backup of working configurations
- Document any custom settings you create
- Check for updates regularly
