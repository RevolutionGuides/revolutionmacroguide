# Settings Tab (6)

## Macro

### Appearance

| Setting | Description |
|---|---|
| Theme | Dark Mode ofc |
| Always on Top | Whether or not Revolution is displayed while you macro |

### Hotkeys
Configurable to the users liking

### Captures

| Setting | Description |
|---|---|
| Fallback Capture Mode | The capture method used when the default mode fails. This controls how the macro takes screenshots of the Roblox game window |
| Detection Frame Rate | How often per second the macro detects buffs and compensates for haste, alongside other checks like backpack percent and health. Lowering this value may reduce CPU usage |

### Misc

| Setting | Description |
|---|---|
| Log Level | Just keep at 0 |
| Load Timeout | Maximum seconds before aborting loading AI models (Recommended: 30 seconds). If you have a lower budget device, set to 60 |

---

## Game

### Movement

| Setting | Description |
|---|---|
| Player Movespeed | Set to your Bee Swarm Character's speed. Verify no speed buffs are active when updating your Movespeed. Common cause for macro problems |
| Camera Sensitivity | Keep at 0, or press the Eye Icon for Auto Detect |
| Key Delay | Default at 20ms, set this higher to 30-50 if you have low FPS |

### Routing

| Setting | Description |
|---|---|
| Use High Alignment | When enabled, the macro disables glider use and aligns to corners and walls, using safer alternative routes in some cases |
| Disable Red Cannon | Macro will avoid using red cannon and instead will walk around the map |

### Join

| Setting | Description |
|---|---|
| Preferred Hive Slot | (6 - 5 - 4 - 3 - 2 - 1), with 1 being closest to the red cannon |
| Hive Claim Method | The method used when claiming a hive. When using detection, the macro will determine what slots are available from the spawn. When using check, the macro will check each hive for availability. |
| Private Server Link | Enter your Private Server Link here. If left empty, will join random public servers |
| Fallback to Public Server | Whether the macro should join a public server or keep retrying if it fails to rejoin a Private Server |

## Networking

This is where you set up your relays for Vic Hop Searchers, Tad Sync, RBC Gather Fuzzy Alts and more

| Step / Setting | Description |
|---|---|
| Start your Relay | Start your Relay |
| Add Relays | Add them clicking the "+" on Available Relays |
| Connect Relays | Find them in wherever you want to setup the Alts/Searchers, and connect them |
| Auto Start/Connect | Auto starts and connects Relays |
