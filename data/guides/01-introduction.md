## Overview of Tabs (6 total)

1. **Gather Tab** - Configure gathering patterns and settings
2. **Collect Tab** - Manage honey collection and conversion
3. **Planters Tab** - Set up planter automation
4. **Status Tab** - Monitor macro performance and stats
5. **Tools Tab** - Access utility functions
6. **Settings Tab** - General macro configuration

Each tab has its own specific settings and configurations tailored to different aspects of automation in Bee Swarm Simulator.
