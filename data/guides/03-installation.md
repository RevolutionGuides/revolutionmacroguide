# Installation Guide

## Downloading Revolution Macro

1. Visit the [GitHub Releases Page](https://github.com/nosyliam/revolution-macro/releases/latest)
2. Download the latest `RevolutionMacro.exe` file
3. Create a dedicated folder (e.g., `C:\Users\YourName\Downloads\RevolutionMacro`)

## Extraction & Setup

> ⚠️ **Important**: The macro MUST be extracted from the ZIP file. Never run it directly from inside the compressed archive.

1. **Extract** the downloaded ZIP file to your chosen folder
2. Ensure `RevolutionMacro.exe` is **inside** a proper folder (not loose)
3. If you encounter "Access is denied" errors, move the folder to:
   - Desktop, Downloads, Documents, or another standard location
   - Avoid Program Files or protected directories

## First Launch

1. Double-click `RevolutionMacro.exe`
2. The macro will perform an initial setup
3. Allow Windows Firewall access if prompted
4. Configure basic settings in the UI

## Verifying Installation

- The macro window should appear without crashes
- You should see the main configuration tabs
- No error messages about missing dependencies

## macOS Installation

1. Download the macOS release from GitHub
2. Extract the archive
3. Move the app to your Applications folder
4. Grant accessibility permissions:
   - System Settings → Security & Privacy → Accessibility
   - Add Revolution Macro to the allowed apps
   - Restart the macro if permissions were just added

## Troubleshooting Installation

If you encounter issues:

- Ensure your OS is up to date
- Verify Windows/macOS has latest security updates
- Check that you have admin rights to run the program
- See the **Troubleshooting** section for specific error fixes
