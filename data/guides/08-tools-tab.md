## RBC Gather

### General

| Setting | Description |
|------|------------|
| Role | Ensure your role is set correctly. Main is your main, and Fuzzy is for your alt |
| Field Selection | Pick the field you wish to gather in for each field color |
| Quest Completion Threshold | Threshold for how much of your RBC quest you have completed at which the macro will automatically move your alt to the next field. If set to 0, quest detection is disabled |
| Field Navigation Mode | Determines how alts move to the next quest field. Manual lets you choose the field. Closest moves to the nearest field. Easy-Hard and Hard-Easy move based on quest difficulty order |

### Alt

| Setting | Description |
|------|------------|
| Rejoin Server on Reset | When enabled, the macro rejoins the server when a reset is needed, allowing your fuzzy alt to recover faster if it fails to walk to the next field |
| Disable Reset when Gathering | When enabled, the alt will attempt to navigate to the next field without resetting |
| Maximum Alts on Field | The maximum number of alts the macro will place on a single field when using automatic quest detection |
| Detect Quests after Round | The minimum round before the macro starts automatically detecting quests and moving alts to quest fields |

### Hotkeys
Customizable to the user

### Field Picker
Connect to a relay to select an alt. This is done in Settings Tab in Networking

## Vic Hop

### General

| Setting | Description |
|------|------------|
| Role | Main kills Vicious, Searcher searches for Vicious bees for Main to kill |
| Server Hop | Whether or not the macro should server hop. Searchers hop regardless, so ensure your roles are set correctly |
| Hop Timeout | The maximum time (in seconds) the macro waits for Roblox to join the next server before force-closing and reopening it |
| Stats | Below are your Vic Hop stats |

### Main

| Setting | Description |
|------|------------|
| Claim Hive when Searching | When enabled, the player will claim hive before searching. When disabled, the macro will need to reset and claim a hive after a Vicious Bee is found |
| Join New Battles | Whether or not the macro should join Vicious Bee battles activated by other players (Recommended to enable) |
| Join In-Progress Battles | Whether or not the macro should join Vicious Bee battles already in progress (Recommended to enable) |
| Vicious Bee Minimum Health | The minimum health percentage that a Vicious Bee must have before joining an in-progress battle |

### Searcher

| Setting | Description |
|------|------------|
| Skip Spider/Pepper | (Recommended to disable) |
| Join Empty Servers | (Recommended to disable) |
| Re-open Roblox After | (Recommended: 2 Hours) |
